# Digital Accessibility Audit

This folder contains an accessibility evaluation case study
focused on public-facing website pages.

## Scope
- Public website pages (navigation, content pages, forms)
- WCAG 2.2 Level AA conformance evaluation
- Keyboard-only, screen reader, and visual testing

## Testing Focus
- Keyboard navigation, focus order, and focus visibility
- Screen reader structure and announcements (landmarks, headings, names/roles/states)
- Forms, instructions, and error handling
- Menus, links, buttons, and other interactive components
- Text and non-text contrast, focus indicators, and reflow at increased zoom
